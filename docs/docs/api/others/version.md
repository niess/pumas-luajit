# Version
_Version of the PUMAS C library._

## Content

A `string` containing the version of the PUMAS C library encoded as
`'(major).(minor)'`, e.g `'1.0'` for PUMAS v1.0.
{: .justify}

## Examples

``` lua
-- Get the C library version
local version = pumas.version
```
